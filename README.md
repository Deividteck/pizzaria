# 🍕 Projeto Pizzaria

Sistema web desenvolvido para gerenciamento de uma pizzaria, permitindo organização de pedidos, produtos e clientes de forma simples e eficiente.

---

## 🚀 Funcionalidades

* 📋 Cadastro de produtos (pizzas, bebidas, etc.)
* 🛒 Gerenciamento de pedidos
* 👤 Cadastro de clientes
* 📊 Visualização de informações
* 🔐 Sistema básico de navegação

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Vite (ambiente de desenvolvimento)

---

## 📁 Estrutura do Projeto

```
project/

```

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/Deividteck/pizzaria.git
```

2. Acesse a pasta do projeto:

```
cd pizzaria
```

3. Instale as dependências:

```
npm install
```

4. Execute o projeto:

```
npm run dev
```

5. Abra no navegador:

```
http://localhost:5173
```
